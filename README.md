# AJ PRO RÉNOVATION

Outil de gestion de chantier conçu pour une utilisation sur tablette.

Application web autonome (un seul fichier HTML) permettant de créer et gérer des fiches de chantier directement depuis le terrain.

## Utilisation

Ouvrir `index.html` dans un navigateur, ou consulter la version en ligne déployée sur Vercel.

## Stack

- HTML / CSS / JavaScript pur
- Aucune dépendance, aucun build
- Hébergement statique (Vercel)
